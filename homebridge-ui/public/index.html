Devices

The Daikin units this plugin controls. The network is scanned automatically when this page opens; units found on the network appear in the list below.

▸ Secure adapter keys (BRP072C)

Most units do not need this and are detected automatically. Only units paired with the Daikin Comfort Control app (secure BRP072C-style adapters, marked 🔒 in the device list) need their 13-digit key, printed on the sticker on the adapter or unit (next to SSID and MAC). The IP must match the entry in Devices exactly.
