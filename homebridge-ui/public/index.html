Devices

The Daikin units this plugin controls. Scan the network to add them without typing, or enter an IP address manually.

▸ Secure adapter keys (BRP072C)

Most units do not need this and are detected automatically. Only units paired with the Daikin Comfort Control app (secure BRP072C-style adapters) need their 13-digit key, printed on the sticker on the adapter or unit (next to SSID and MAC). If the Homebridge log shows a secure BRP072C-style adapter hint for an IP, add that IP and its key here. The IP must match the entry in Devices exactly.
