Device finder

Searches the local network for Daikin units (UDP broadcast, ~4 seconds) so you can add them without typing IP addresses. If Homebridge runs in a Docker bridge network the broadcast cannot reach your LAN — use host networking or enter the IP manually.

▸ Secure adapter keys (BRP072C)

Most units do not need this and are detected automatically. Only units paired with the Daikin Comfort Control app (secure BRP072C-style adapters) need their 13-digit key, printed on the sticker on the adapter or unit (next to SSID and MAC). If the Homebridge log shows a secure BRP072C-style adapter hint for an IP, add that IP and its key here. The IP must match the entry in Climate IPs exactly.
